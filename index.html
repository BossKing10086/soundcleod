What's SoundCleod?

SoundCleod is a Mac OS X application which brings SoundCloud to the desktop without using a browser.

Why is that good?

  • No more 'trying to find in which tab is SoundCloud running'..
  • Media keys support! (Yeah, those with ▶ ❚❚ on them.)
  • Stops on suspend.

How to get it?

  • Download from here
  • Open the zip file.
  • Move SoundCleod to the Applications folder.
  • Go to Applications, find SoundCleod.
  • Double click SoundCleod.
  • If it greets with a prompt about an 'unidentified developer', do the following steps (why is that?):
    • Right click (or Control-click) SoundCleod
    • Select Open from the menu
    • Click Open in the dialog box.
    • If prompted, enter an administrator name and password.
